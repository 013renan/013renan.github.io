Sobre o Moodle
O Moodle é um sistema de gerenciamento de aprendizado baseado na web, usado para criar sites de cursos online. É um software de código aberto, desenvolvido sob a licença GNU General Public License. O Moodle é usado em todo o mundo para ensinar on-line, com mais de 100 milhões de usuários em mais de 200 países.
